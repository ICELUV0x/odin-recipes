# Odin Recipes

A simple recipe website built with HTML. Created as part of [The Odin Project](https://www.theodinproject.com/) Foundations curriculum.

## Recipes

- BBQ Ribs
- Chicken Quesadilla
- Caramel Apple Cheesecake

## What's inside

- Main page with links to all recipes
- Each recipe page has a photo, description, ingredients, and step-by-step instructions
- Navigation back to home on every page

## Built with

- HTML
